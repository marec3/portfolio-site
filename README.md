# Personal Portfolio & Blog

This project is my personal portfolio and blog website built with Next.js.  
It showcases my experience, blog articles, and learning journey as a student in Automation and Computer Engineering.

The website includes sections for work experience, blog posts, and ways to connect with me.

---

## Features

- Personal portfolio website
- Blog section with articles about my learning journey
- Work experience section
- Responsive design
- Modern UI built with Tailwind CSS

---

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## Blog Articles

The blog section includes articles about my experience and learning process:

- My Journey as an Automation and Computer Engineering Student
- How I Approach Learning New Technologies
- My Work and Travel Experience in the United States
